# sunray_mobile

Flutter-Basis fuer die neue Alfred-App.

## Ziel

- Discovery per mDNS
- Dashboard, Karte, Missionen, Service
- Backend-Preview aus `sunray-core`
- OTA fuer App, Pi und STM spaeter ueber Service-Screen

## Geplante Schritte

1. `flutter pub get`
2. Android/iOS Plattformen anlegen, falls noch nicht vorhanden
3. Discovery anbinden
4. Dashboard und Karteneditor vervollstaendigen

## Hinweis

Dieses Geruest wurde manuell vorbereitet, weil auf der aktuellen Maschine kein
`flutter` installiert ist. Sobald Flutter verfuegbar ist, kann das Projekt
normal initialisiert und erweitert werden.
